# BureauAI – Landing Page

Site vitrine pour BureauAI, SaaS d'outils IA pour les bureaux d'études au Maroc et en Europe.

## Stack Technique

- Next.js 13 (Pages Router)
- React 18
- Tailwind CSS (dark theme personnalisé)
- Framer Motion (animations)
- Jest & Testing Library (tests)
